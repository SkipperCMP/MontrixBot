# LEGACY_UI_INDEX.md  
Legacy UI — Index & Reading Guide

Статус: LEGACY / Reference-only

Этот файл является **точкой входа** в legacy UI-материалы проекта MontrixBot
и задаёт корректную интерпретацию исторических и сторонних UI.

---

## 1. Что такое legacy UI

Legacy UI — это:
- исторические прототипы;
- сторонние торговые интерфейсы;
- UX-эксперименты, созданные до фиксации текущей архитектуры.

Legacy UI:
- сохраняется как справка;
- используется для UX-анализа;
- **не может** быть использован как канон.
