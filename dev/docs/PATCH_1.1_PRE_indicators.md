# MontrixBot 1.1 — PREPATCH: indicators (RSI/EMA/SMA)

Этот патч добавляет модуль `core/indicators.py` с базовыми индикаторами,
которые будут использоваться в версии 1.1 и выше.

Содержит:
  - sma(values, period)
  - ema(values, period)
  - rsi(values, period=14)

Поведение:
  - Патч НЕ меняет существующий UI и ядро.
  - Патч НЕ подключён к графику автоматически.
  - Никаких побочных эффектов при установке патча нет.

Как установить:
  1) Распаковать архив поверх папки проекта MontrixBot_1.0.1_FINAL_SAFE_ON.
  2) Убедиться, что появился файл `core/indicators.py`.
  3) После этого можно в следующих патчах 1.1 добавлять вызовы этих функций
     в графики, стратегии и Telegram-команды.
