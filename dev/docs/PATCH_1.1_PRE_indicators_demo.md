# MontrixBot 1.1 — PREPATCH: indicators demo (RSI/EMA/SMA)

Этот патч добавляет вспомогательный скрипт:

  tools/rsi_demo.py

Назначение:
  - продемонстрировать использование `core.indicators.sma`, `ema`, `rsi`;
  - служить простым ручным тестом в процессе разработки 1.1;
  - подтвердить, что модуль не требует внешних зависимостей и работает
    в текущей среде.

Запуск:

  python tools/rsi_demo.py

Результат:
  - в консоль выводятся 10 последних строк с:
    * синтетической ценой
    * SMA(period)
    * EMA(period)
    * RSI(period)
  - это безопасный, полностью оффлайн-тест, не затрагивающий Binance и UI.

Патч не меняет существующее поведение бота 1.0.1 и может безопасно
накладываться поверх baseline 1.0.1_FINAL_SAFE_ON вместе с патчем
`MontrixBot_1.1_PREPATCH_indicators_rsi.zip`.
