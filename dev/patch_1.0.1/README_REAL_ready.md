
# MontrixBot 1.0.1+ — REAL-ready

В этом патче:
- Исправлен вызов реальных ордеров из UI (надёжный subprocess, абсолютные пути).
- По умолчанию используется `--ask` (SAFE-код запрашивается перед ордером).
- Добавлены консольные хелперы: `scripts/buy.ps1` и `scripts/buy.cmd`.

## Применение
Распаковать в корень `MontrixBot_1.0.1` с заменой.

## Проверка
- Удалить/переименовать `SAFE_MODE`.
- Запустить `python ui/app_step8.py` и нажать **Buy (Market)** — появится запрос SAFE-кода и JSON-ответ.
